发布到 Github Pages

SSH Keys

任意文件夹右键菜单选择 Git Bash Here

输入以下命令：ssh-keygen -t rsa -C "你的GitHub注册邮箱"

敲几次回车后，找到 id_rsa.pub

点自己的 Github 头像 —— Settings —— SSH and GPG keys —— New SSH Key

将 id_rsa.pub 的内容全部复制并粘贴到 KEY 文本框里，再点击 Add SSH Key

完成后在终端输入 ssh git@github.com 验证是否连接成功

部署

在 Hexo 根目录的 _config.yml 文件中配置 deploy

如果 https 链接在后面出现问题，则选择 SSH 链接

在博客根目录下启动 Git Bash，输入命令安装 hexo-deployer-git

接着使用 hexo clean 清理缓存，使用 hexo d 部署到 Github 上
