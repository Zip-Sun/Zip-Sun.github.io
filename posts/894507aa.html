在2021年使用 Windows XP

发表于2020-07-06|更新于2023-02-26
|字数总计:1.8k|阅读时长:6分钟|阅读量:

官方支持情况：
主要支持：已于2009年4月8日到期
扩展支持：已于2014年4月8日到期

实机环境：联想家悦E2650(Pentium E5300 2.6GHz、ATI Radeon HD4350、DDR2 2G内存)

使用镜像：zh-hans_windows_xp_professional_with_service_pack_3_x86_cd_vl_x14-74070.iso

如果要在实机重装系统，必须要：

  1. 在安装前备份重要文件。
  2. 从控制面板内检查系统当前安装的程序。
  3. 确保其他设备（如打印机）的驱动程序能够在安装后恢复（最好是保有驱动的安装包）。

注：本文以下部分链接引自互联网档案馆。

补丁相关

Windows Update仍可使用，但升级到Microsoft Update的网页无法跳转，需要直接打开这个网站。

Windows Update与Microsoft Update均已不可使用。建议使用卡饭论坛整合补丁或者雨林木风的整合补丁。

此外自2014年后微软提供的几个XP补丁需要手动下载，可查阅以下网址。

有度盘会员可以直接下载整合包：【全套续命补丁在此】Windows XP SP3后续更新集合（简中）（2020年5月20日）-远景论坛-微软极客社区 (pcbeta.com)

此外国外大神提供了 Legacy Update 插件代替自带的 Windows Update。（尚不清楚对简体中文补丁的支持如何）

个人认为几个很重要的（如果不想安装一堆补丁）

KB4012598（Wanacry漏洞的紧急补丁）

KB4500331（2019年发布的高危安全更新）

KB942288（Windows Installer 4.5的更新补丁）

KB955704（使XP支持exFAT格式）

cn_windows_internet_explorer_8_for_windows_xp_x86.exe（IE8 for XP）

kb4316682和kb4019276（在XP中追加对 TLS 1.1 和 TLS 1.2 支持）（需要伪装WES或PosReady，以及修改注册表开启。）

IE8补丁

用于 Windows XP SP3 的 Internet Explorer 8 累积安全更新程序 (KB4018271)

可选补丁

KB925850（WMP11下载）

VistaFont_CHS.EXE（微软雅黑，配合下面的ClearType能达到更好的阅读效果）

ClearType Tuner PowerToy（启用ClearType）

Office相关

个人只推荐 Office 2010，其他版本过于古老且都需要格式兼容包。

2003

镜像：sc_office_2003_pro.iso

kb923618（SP3补丁包）

kb924074（格式兼容包，需要SP3补丁）

kb2526297（格式兼容包的SP3补丁，需要[KB942288]）

2007

镜像：cn_office_enterprise_2007_DVD_VL_X12-19567.iso

镜像：zh-hans_office_standard_2007_x86_cd_x12-18931.iso（包含word、excel、PowerPoint、outlook）

kb2526086（SP3补丁包）

kb924074（格式兼容包）

kb2526297（格式兼容包的SP3补丁，需要[KB942288]）

相关新闻： 《微软：结束对Excel和PowerPoint查看器以及Office兼容包的支持》

2010

镜像：SW_DVD5_Office_Professional_Plus_2010w_SP1_W32_ChnSimp_CORE_MLF_X17-76734.iso

kb2687455（SP2补丁包）

可用软件

标为“版本”即官方后续不再为XP提供更新版本，标为“当前版本”即截至本文更新时官方仍在维护的版本。

运行库

.net Framework

下载地址：

dotnetfx35SP1.exe（包含从2.0到3.5SP1的内容）

dotNetFx40_Full_x86_x64.exe

注：.net framework 4.0之后的版本不再支持XP。

DirectX

下载地址：微软官方下载

或者使用DirectXRepair，可以连带将C++运行库一起安装。

Flash Player

当前版本：32

建议使用29版本，此为最后没有地区限制的版本。

Adobe 官方已经取消下载，推荐使用 Clean Flash Player，在吾爱论坛可以下载。

XNA Framework

下载地址：微软官方下载

部分软件需要。

Java

版本：jre-7u80-windows-i586.exe

版本：jdk-8u25-windows-i586.exe（推荐）

Python

版本：3.4.4

浏览器

Chrome

版本：49.0.2623.112

也可以选择直接下载官方离线版或者绿色版（由PortableApps提供）

Firefox

版本：52.9.0 ESR

可以安装旧版本扩展。

Mypal

当前版本：29.1.1

基于Pale Moon（Firefox内核）的开源浏览器，可安装中文语言包。

安装中文语言包的方法：语言包拖入浏览器，安装后在地址栏输入about:config，同意警告后搜索general.useragent.locale，修改参数为zh-CN，之后重启浏览器。

压缩工具

Bandizip

版本：5.23

压缩文件管理工具，可提供高速的解压缩速度和其他各种实用功能。

7-Zip

当前版本：19.00

开源的压缩软件，可以调节的压缩参数比较多，缺点是界面简陋。

多媒体

千千静听

个人推荐版本：V5.7.9

需要歌词功能可以更新歌词服务器。

K-Lite Codec Pack

版本：K-Lite_Codec_Pack_1385_Basic.exe

视频解码器，直接安装Basic版可搭配系统自带的WMP使用。

PotPlayer

个人推荐版本：201209(1.7.21396)

多媒体播放器。该版本是最后支持SWF格式的版本。

IrfanView

当前版本：4.5.7

看图软件，支持绿色安装。语言包和更多支持格式需要单独下载。

注：目前发现打开webp文件报错，暂无解决办法。

建议在设置里勾选“仅载入关联文件”以免打开同文件夹内的pdf等文件。（如果想用它来看PDF文件还需要额外安装Ghostscript）

Adobe Photoshop

版本：CS6 13.0.1(13.1.3及以后不再支持)

如果不是非常需要，个人推荐Photoshop 7.0版。

Adobe Premiere Pro

版本：CS4（参考链接）

After Effects

版本：CS4（参考链接）

输入法

谷歌拼音输入法

版本：2.7.23.124

已经被废弃，但仍可以通过官网下载。

华宇（紫光）拼音输入法

新版本：7.1.3.150

旧版本：unispim6.9.1.183.exe

老牌输入法，支持输入界面大字体。

QQ拼音输入法

个人推荐版本：V4.7

安装后关闭更新检查可以正常使用，支持云端联想、无障碍版皮肤。

搜狗拼音输入法

医生版：V1.1

办公

微信WeChat

版本：3.2.1

Adobe Reader

版本：11（XI）

PDF阅读器，不想安装可以选择直接用Chrome打开PDF文件。

注：该版本存在一个闪退Bug，可通过上面链接中的文章解决。

数科OFD阅读器

当前版本：？？？

用于打开OFD文件，也支持OFD转换PDF。

优效日历

当前版本：2.1.3.15

win10风格的桌面日历，支持农历、天气、待办事项等多种功能。

Snipaste

当前版本：v1.16.2 (2018.01.22)

截图工具，也可以用来贴图。

Thunderbird

版本：38.5.0

Mozilla制作的电子邮件客户端（相关帮助）。可以替换自带的Outlook Express。

LibreOffice

版本：5.4.7.2

代替Office，非开源软件爱好者不推荐使用。

开发

Geek Uninstaller

当前版本：1.4.7.142

卸载软件用，附带清理注册表的功能。

HashTab

版本：5.2.0.14

在文件属性中查看文件哈希值。

IObit Unlocker

版本：1.2.0

解除文件占用。

DVDFab Virtual Drive

版本：1.5.1.1

虚拟光驱，用于挂载iso文件。

Process Explorer

下载地址：Microsoft Docs

可用来代替系统自带的任务管理器。

FileZilla Client

版本：3.8.0

FTP客户端。此为最后支持XP的版本。

Folder Size

当前版本：2.6

在资源管理器中可查看文件夹大小，在系统的控制面板中能够详细配置。

Eclipse

版本：Photon(2018 June)

最后支持XP的版本。

Notepad++

版本：7.9.2

文本编辑软件。由于该软件开发者支持GD，目前正在寻找可以替代的软件。

Notepad2

当前版本：v4.23.01r4584

支持 XP，且可以替换系统自带的记事本。
