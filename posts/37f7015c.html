在 2023 年使用 Windows 8.1

发表于2023-02-08|更新于2023-03-02
|字数总计:980|阅读时长:3分钟|阅读量:

本文大部分推荐软件也可适用于 Windows 7。

前言

2023 年 1 月 10 日，微软停止了对 Windows 8.1 的支持。我也正式决定把自己用了 4 年的 Win10 重新装回最初的 Win8.1, 这台“坠机堡垒”陪我走到今天也确实尽力了。给她换回旧系统好好养老吧。

更新

Windows Update 仍能正常工作，第一次扫描需要较长时间。

Metro 应用

自带应用商店仍能使用，但国区大部分 App 的官方已不再更新，自带的几个也无法正常运行，正在寻找替代方案。

推荐软件

标为“版本”即官方后续不再为 8.1 提供更新版本，标为“当前版本”即截至本文更新时官方仍在维护的版本。

OneDrive

安装后会自动转为 Business 版，并提示在 8.1 系统使用自带版本。但因为 Metro 无法更新，App 会提醒更新系统，陷入死循环。

可以使用组策略（计算机配置——管理模板——Windows 组件——OneDrive——禁止使用OneDrive进行文件存储）关闭。

更新：微软官方已经终止支持

Logitech G HUB

版本：2021.3.9205

点击网页左边的小箭头选择安装系统。

联网下载，所以无论你用多老的安装包都会安装到上面这个版本。

能正常安装，但会提示罗技驱动没有数字签名无法安装。尝试多个方法仍不能解决。

Aero Glass

版本：1.4.6

为 Win8.1 开启 Aero 效果，推荐 1.4.6 版，缺点是开机弹窗和桌面有水印。（网上的破解都是针对1.2.5版，但该版本是针对Win8.1早期版本，安装到现在的 8.1 系统会有很多 Bug）

Clipdiary

当前版本：5.7

可以查看剪贴板历史记录，支持快捷键启动，可惜不能绑定 Win + V 键。

开始下载的是 60 天试用版，点击 帮助→激活免费版本 将跳转官网提供免费版激活码。（暂未发现付费版与免费版的区别）

CopyQ

当前版本：6.4.0

也是剪贴板扩展，但是是开源软件。

一些小技巧

Metro 应用关闭后仍然在任务管理器里看到？

Win8.1 的内存机制，当 Metro 应用关闭时并非传统的关闭，而是后台挂起。在任务管理器——查看——状态值——显示已挂起状态，打勾后可知。

如果想要完全关闭，请在应用界面按 Alt + F4 ，或者尝试以下方案。

BUG

Windows 事件 ID 3

会话“Circular Kernel Context Logger”已停止，原因是存在以下错误: 0xC0000188

根据日志详细信息定位，出问题的文件是 BootCKCL.etl（启动跟踪日志）

暂未找到解决方案，不影响正常使用。

Windows 事件 ID 1000

错误应用程序名称: pcsx2-qtx64-avx2.exe，版本: 1.7.3335.0，时间戳: 0x63305b03
错误模块名称: api-ms-win-core-memory-l1-1-6.dll，版本: 6.3.9600.20718，时间戳: 0x636f3964

基于 Qt 6.4 的应用需要 Windows 10 以上的系统才能正常运行。建议更换旧版本。

Windows 事件 ID 10016

应用程序-特定 权限设置并未向在应用程序容器 不可用 SID (不可用)中运行的地址

修复方法：https://pcrepair.w-pickup.com/post-5381/

出现问题的原因是缺少访问权限，不影响正常使用，可以无视。

Windows 事件 ID 36887

从远程终点接收到一个严重警告。TLS 协议所定义的严重警告代码为 70。

修复方法：https://pkiwithadcs.com/details_about_36887_36888/

根据方案，代码 70 是因为在 TLS 握手期间，客户端与服务器要求的版本不一致。

推测可能是 Win 8.1 版本低的问题，暂未找到解决方案。

不影响正常使用。
