在2020年使用 Windows XP

官方支持情况：
主要支持：已于2009年4月8日到期
扩展支持：已于2014年4月8日到期

文章最后更新日期：2020/10/24

测试环境：VMWare Workstation 14.x虚拟机 单核+512M内存
使用镜像：zh-hans_windows_xp_professional_with_service_pack_3_x86_cd_vl_x14-74070.iso
切记：如果要在实机重装系统，①必须在安装前备份重要文件，②从控制面板内检查系统当前安装的程序，③从设备管理器确保其他设备（如打印机）的驱动程序能够在安装后恢复。

补丁相关

考虑到兼容问题，本文不提及Windows Embedded POSReady 2009相关补丁。

Windows Update与Microsoft Update已不可使用。建议使用雨林木风的整合补丁。

此外自2014年后微软提供的几个XP补丁需要手动下载，可查阅以下网址。

个人认为很重要的

KB4012598（Wanacry漏洞的紧急补丁）

KB4500331（2019年发布的高危安全更新）

KB942288（Windows Installer 4.5的更新补丁）

KB955704（使XP支持exFAT格式）

cn_windows_internet_explorer_8_for_windows_xp_x86.exe（IE8 for XP）

IE8补丁

用于 Windows XP SP3 的 Internet Explorer 8 累积安全更新程序 (KB4018271)

可选补丁

kb925850（WMP11下载）

VistaFont_CHS.EXE（微软雅黑）

ClearType Tuner PowerToy（启用ClearType高级设置）

Office相关

2003

镜像：sc_office_2003_pro.iso

kb923618（SP3补丁包）

kb924074（格式兼容包，需要SP3补丁）

kb2526297（格式兼容包的SP3补丁，需要[KB942288]）

2007

镜像：cn_office_enterprise_2007_DVD_VL_X12-19567.iso
镜像：zh-hans_office_standard_2007_x86_cd_x12-18931.iso（包含word、excel、PowerPoint、outlook）

kb2526086（SP3补丁包）

kb924074（格式兼容包）

kb2526297（格式兼容包的SP3补丁，需要[KB942288]）

相关新闻： 《微软：结束对Excel和PowerPoint查看器以及Office兼容包的支持》

2010

镜像：SW_DVD5_Office_Professional_Plus_2010w_SP1_W32_ChnSimp_CORE_MLF_X17-76734.iso

kb2687455（SP2补丁包）

可用软件

标为''版本''即官方后续不再为XP提供更新版本，标为''当前版本''即截至本文更新时官方仍在维护的版本。

.net Framework

支持安装至.net4.0

下载地址：https://dotnet.microsoft.com/download/dotnet-framework

dotnetfx35SP1.exe（包含从2.0到3.5SP1的内容）

dotNetFx40_Full_x86_x64.exe

DirectX

下载地址：https://www.microsoft.com/zh-CN/download/details.aspx?id=35

或者使用DirectXRepair，可以连带将C++运行库一起安装。

Java

版本：jre-7u80

jre-7u80-windows-i586.exe

版本：jdk-8u25

jdk-8u25-windows-i586.exe

eclipse

版本：Photon(2018 June)

Chrome

版本：49

Firefox

版本：52.9.0 ESR

可以安装旧版本扩展。

Mypal

当前版本：28.14.2

基于Pale Moon（Firefox内核）的开源浏览器，可安装中文语言包。

安装中文语言包的方法：语言包拖入浏览器，安装后在地址栏输入about:config，同意警告后搜索general.useragent.locale，修改参数为zh-CN，之后重启浏览器。

K-Lite Codec Pack

版本：13.8.5

K-Lite_Codec_Pack_1385_Basic.exe

视频解码器，直接安装Basic版可搭配系统自带的WMP使用。

谷歌拼音输入法

版本：2.7.23.124

已经被废弃，但仍可以下载。

（紫光）华宇拼音输入法

版本：V6.9

unispim6.9.1.183.exe

老牌输入法，支持输入界面大字体。

原官网已更新，当前链接可能随时被作废。

优效日历

当前版本：2.0.10.16

win10风格的桌面日历，支持农历、天气、待办事项等多种功能。

Adobe Flash Player

当前版本：32

29版本为最后没有地区限制的版本。

Adobe Photoshop

版本：CS6 13.0.1(13.1.3及以后不再支持)

Adobe Premiere Pro

版本：CS4（参考链接）

After Effects

版本：CS4（参考链接）

Bandizip

当前版本：5.23

LibreOffice

版本：5.4.7.2

HashTab

版本：5.2.0.14

在文件属性中查看文件哈希值。

Thunderbird

版本：38.5.0

Mozilla制作的电子邮件客户端，相关帮助。可以替换自带的Outlook Express。
